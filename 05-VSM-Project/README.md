# Project 05 - Value Stream Mapping (VSM) Project

## Project Overview

This project focuses on Value Stream Mapping analysis for optimizing manufacturing processes. It includes comprehensive documentation and data analysis for identifying and eliminating waste in production systems.

## Project Information

**Customer:** Valeo India Pvt. Ltd, Pune
**Product:** Brush Holder
**Raw Material:** PPS GF 40 Grade (Ryton R4 200)
**OEM:** Suzuki

## Annual Volumes

- Year 1-7 (2020-2026): 600,000 units per year

## Key Files

- **[VSM Presentation On Brush Holder.pptx](VSM%20Presentation%20On%20Brush%20Holder.pptx)** - Detailed presentation on the VSM analysis- **[VSM Process Flow Base Unit.ppt](VSM%20Process%20Flow%20Base%20Unit.ppt)** - Process flow diagram and operations
- **[VSM DATA.xlsx](VSM%20DATA.xlsx)** - Supporting data and analytics
## Key Metrics

**Standard Takt Time:** 37.762 sec/part
**Actual Takt Time:** 37.951 sec/part
**Average Monthly Volume:** 39,250-39,769 units (Pune), 10,066-10,800 units (Mexico)

## Areas of Improvement

1. **Operator Motion for Material Movement** - Streamline material handling processes
2. **Gauge Test and Verification** - Optimize inspection procedures
3. **Change-Over Time Reduction** - Current: 60 minutes, Target: 15-30 minutes
4. **Bin Storage Management** - 89 bins stored at machine station - requires DOC relocation for safety

## Wastes Identified (Lean Manufacturing)

- **Faster-than-necessary Pace:** Overproduction issues
- **Waiting Time:** Delays in material transportation
- **Conveyance:** Material movement optimization needed
- **Processing Complexity:** Streamline procedures
- **Excess Stock:** Inventory management
- **Unnecessary Motion:** Ergonomic improvements
- **Defects:** Quality control enhancement
