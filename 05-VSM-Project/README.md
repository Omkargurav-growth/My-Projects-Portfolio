# Project 05 - Value Stream Mapping (VSM) Project

## Project Overview

This project showcases a comprehensive **Value Stream Mapping (VSM)** analysis for optimizing manufacturing processes at **Valeo India Pvt. Ltd, Pune**. Value Stream Mapping is a lean management technique used to identify and eliminate waste in production systems, thereby increasing efficiency and productivity.

### Key Objectives
- Analyze the current state manufacturing process
- Identify value-added and non-value-added activities
- Design future state for waste elimination
- Implement continuous improvement initiatives

## Identified Wastes (7 Types of Muda)

1. **Faster-than-Necessary Pace** - Overproduction issues
2. **Waiting Time** - Material transportation delays
3. **Conveyance** - Material handling optimization needed
4. **Processing Complexity** - Unnecessary procedures
5. **Excess Stock** - 89 bins stored at machine station
6. **Unnecessary Motion** - Ergonomic improvements required
7. **Defects** - Quality control enhancement opportunities

---

## Areas of Improvement

### High Priority Improvements

**1. Operator Motion Optimization**
- **Current State:** Unnecessary movements in material handling
- **Target:** 15-20% reduction in motion waste
- **Expected Impact:** 5-7 seconds per cycle savings
- **Implementation Timeline:** 2-3 weeks

**2. Changeover Time Reduction**
- **Current State:** 60 minutes per changeover
- **Target State:** 15-30 minutes
- **Potential Savings:** 30-45 minutes per changeover
- **Annual Impact:** 2-3 hours saved per changeover series
- **Expected Capacity Gain:** 5-7%

**3. Material Storage Optimization**
- **Issue:** 89 bins stored at machine station
- **Risk:** Safety hazards, blocked floor access
- **Solution:** Relocation to Day-Old-Concept (DOC) area
- **Benefits:** Better space utilization, improved safety

**4. Quality & Verification Process**
- **Current:** Manual gauge testing delays production
- **Improvement:** Automated/semi-automated verification
- **Expected Gain:** 3-5% capacity increase
- **Quality Improvement:** Reduced defects
